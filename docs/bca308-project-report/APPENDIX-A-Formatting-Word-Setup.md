# Appendix A — BCA 308 Report Formatting (Word Setup Checklist)

> Use this when **pasting** from `.md` files into **Microsoft Word** (or WPS / LibreOffice with equivalent options).

## A.1 Page setup

| Setting | Value (as per your notice) |
|---------|----------------------------|
| Left margin | **3.0 cm** |
| Right margin | **2.0 cm** |
| Top margin | **2.54 cm** |
| Bottom margin | **2.54 cm** |
| Page numbers | **Bottom center** (Insert → Page Number) |

*Path in Word: Layout → Margins → Custom Margins.*

## A.2 Body text

| Item | Setting |
|------|--------|
| Font | **Times New Roman** |
| Size | **12 pt** |
| Line spacing | **Double** |
| Alignment | **Justified** |
| Paragraph spacing | **6 pt** before and after (Paragraph → Spacing) |

## A.3 Headings (match your internal guide; typical pattern)

| Level | Font | Size | Other |
|-------|------|------|--------|
| **Chapter** (e.g. CHAPTER 1) | Times New Roman | **20 pt** | **Center**; 30 pt spacing after (as per your sheet) |
| **Section** (1.1 Introduction) | Times New Roman | **14 pt** | **Underlined**; **Left**; 12 pt spacing (as per your sheet) |
| **Sub section** (optional) | 12 pt bold or 12 pt with numbering | TNR 12 | Consistent with department |

*Your notice: “Paragraph Heading: 14, Underlined, Left aligned, 12pt spacing” and “Chapter Heading: 20, Center aligned, 30pt spacing” — apply exactly.*

## A.4 Code and code captions

- Code in report: **Courier New**, **10 pt**.  
- **Single-spacing** for code blocks is common; confirm with guide.  
- **Caption** below: *Code 1.1 — Main Application Class* (italicize label if your college style says so).

## A.5 Figures and tables

- **Figure** caption: usually **below** the image (*Fig. 1.1 — DFD Context Diagram*).  
- **Table** caption: **above** the table (*Table 1.1 — User requirements*).  
- List of figures / list of tables — add if your department requires.

## A.6 Page count (~75 pages excluding code)

- **Excluding coding** = main narrative; put **lengthy code** in an **appendix** and/or **separate code book** as instructed.  
- This Markdown pack is a **skeleton** — add **screenshots**, **expanded theory**, and **viva Q&A** to grow pages.  

## A.7 Table of contents

- Word: **References → Table of Contents → Automatic**.  
- Update after final edits.  

## A.8 Print & binding

- **One** summary/abstract (synopsis) as per notice.  
- **One** hard copy of the main report + certificate + acknowledgement + college rules.  
- **Spiral / hard bound** as per your department (confirm with staff).
